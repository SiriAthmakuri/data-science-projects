# Credit Card Fraud Detection

## Objective
Detect fraudulent credit card transactions using machine learning models.

## Dataset
- `8.Credit_Fraud_Class_Prediction.ipynb` → Jupyter Notebook with analysis and model  
- `CREDIT CARD FRAUD PREDICTION.docx` → Project report  
- `dataset.csv` → Dataset used (if available)  

**Target Variable:**  
- `Class` → 0: Legitimate transaction, 1: Fraudulent transaction  

---

## Steps / Workflow
1. **Data Preprocessing:** Handling missing values, feature scaling, and data cleaning  
2. **Exploratory Data Analysis (EDA):** Understanding data distribution, checking class imbalance  
3. **Model Building:** Logistic Regression, Random Forest, and other classification models  
4. **Model Evaluation:** Accuracy, Precision, Recall, F1-score, ROC-AUC score  
5. **Feature Importance Analysis:** Identifying which features contribute most to fraud detection  

---

## Tools & Libraries
- Python  
- Pandas, NumPy  
- Scikit-learn  
- Matplotlib, Seaborn  
- Jupyter Notebook  

---

## Usage
Open the notebook `8.Credit_Fraud_Class_Prediction.ipynb` to explore the analysis and models.  
You can run the notebook step-by-step to reproduce results.
