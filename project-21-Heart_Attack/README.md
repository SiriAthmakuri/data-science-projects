# Heart Attack Prediction

## Objective
Predict the likelihood of a heart attack based on patient health data.

## Dataset
- Target: `output` (0 → No Heart Attack, 1 → Heart Attack)
- Features: age, blood pressure, cholesterol, etc.
- Binary classification problem

## Steps
1. Data preprocessing (missing values, encoding, scaling)
2. Exploratory Data Analysis (EDA)
3. Train-test split & cross-validation
4. Model building: Logistic Regression, Random Forest, SVM
5. Model evaluation: Accuracy, Precision, Recall, F1-score, ROC-AUC
6. Feature importance analysis

## Tools & Libraries
- Python, Pandas, NumPy
- Scikit-learn, Matplotlib, Seaborn
- Jupyter Notebook

## Usage
Open `heart_analysis.ipynb` to explore the analysis and models.
